Project idea #1

Smart Vacuum Robot

Possible Kick-off

2021

Current project account balance

7.5m Lira

Project overview

Description

Smart Vacuum Robot is a fully automatic and artificially intelligent vacuum robot, which is able to vacuum up to 1,500 sq ft at a time. Its innovative feature is a mapping function through which the robot can be sent to any place via an app.


CleverClean Inc. has successfully launched a similar vacuum robot last year, but without the mapping function and the long lasting battery. Based on the profound experience with the previous vacuum product and the available market surveys, the returns for Smart Vacuum Robot will be modest but very certain.

To this day, there is no serious competitor in the field of vacuum robots and a market analysis does not reveal any indications on competitors planning to develop a smart vacuum robot.

Initial investment

The initial investment to start the Smart Vacuum project amounts to 3.5m Lira.

Project idea #2

Smart Mob Robot

Possible Kick-off

2021

Current project account balance

7.5m Lira

Project overview

Description

Smart Mob Robot is a fully automatic and artificially intelligent mob robot, which is able to vacuum and mop. However, due to space restrictions only a smaller battery can be used, so up to 1,000 sq ft can be cleaned at a time. Smart Mob Robot also has an innovative mapping function through which the robot can be sent to any place via an app.


Due to operating in an unknown market, it is possible that there will be no market demand at all for Smart Mop Robot. However, if the company succeeds in convincing customers of this innovation, it could generate high sales and a large market share as a "first mover".

To this day, there is no serious competitor in the field of mob robots and a market analysis does not reveal any indications on competitors planning to develop a smart mob robot.

Initial investment

Due to its smaller battery, the initial investment to start the Smart Mob Robot project also amounts to 3.5m Lira.
